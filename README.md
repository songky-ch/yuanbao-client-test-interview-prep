# 备战元宝客户端测试面试

这是一个用于沉淀元宝客户端测试面试准备内容的 Obsidian 知识库。

## 使用方式

- 用 Obsidian 打开本仓库目录。
- 从 [[00-索引/知识库首页]] 开始查看。
- 新增训练记录放入 `04-面试训练/`。
- 新增复盘内容放入 `05-复盘/`。
- 项目、测试专题、AI Agent 测试相关材料按目录归档。

## 当前结构

- `00-索引`: 知识库入口和更新规则
- `01-面试计划`: 阶段计划与每日推进
- `02-项目梳理`: 项目包装、业务理解、链路图
- `03-测试专题`: 客户端测试、AI Agent 测试、接口与日志定位
- `04-面试训练`: 模拟面试记录
- `05-复盘`: 错题、薄弱点、表达优化
- `artifacts`: 图示和原始素材
