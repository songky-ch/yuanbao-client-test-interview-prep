# AI Agent 测试专题

## 测试对象

- 意图识别。
- 多轮上下文。
- Prompt 规则。
- 工具调用。
- 知识库检索。
- 权限边界。
- 结果生成。

## 核心验证点

- 用户问题是否被正确分类。
- 是否调用了正确工具。
- 工具参数是否正确。
- 工具结果是否被正确引用。
- 无权限数据是否被拦截。
- 不确定问题是否拒答或提示核实。
- 回答是否存在事实错误或幻觉。

## 回归重点

- Prompt 变更后检查历史高频问题。
- 工具接口变更后检查参数映射。
- 知识库更新后检查召回和引用。
- 权限策略变更后检查越权访问。

## 面试表达模板

我会把 AI Agent 测试拆成输入理解、任务规划、工具调用、结果生成和安全边界几层。除了看最终回答是否正确, 还会检查它有没有调用正确工具、参数是否准确、是否引用了可信数据、无权限数据是否被拦截, 以及在信息不足时是否能拒答或提示用户补充。
